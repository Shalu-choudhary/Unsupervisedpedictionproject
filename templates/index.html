Project Details

The objective of this project was to predict suitable crops that can be grown on a particular type of soil based on soil quality parameters (such as Nitrogen, Phosphorus, Potassium, pH) and climatic factors (humidity, rainfall). The first step of this project was to preprocess the data using Pandas to make it suitable for our Machine Learning model to get trained to. Then we used Matplotlib and Seaborn to plot different graphs and understand the underlying pattern between the various features and crop suitable. Then we used trained a K Means Clustering model to create clusters for the similar data points. On the basis of the existing labels ( not used for the training purpose ), we identified which crops will grow given the land condition.
